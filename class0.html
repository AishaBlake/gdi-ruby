Let's Develop It

Let's setup our computer for Ruby programming

  • Let's install a text editor - Install Sublime Text 2
  • Install Ruby (This step is for Windows users only. GNU/Linux and MacOSX come with Ruby installed)
  • (Windows only): After installing Ruby, open the "powershell" program and type: 
    [Environment]::SetEnvironmentVariable("Path", "$env:Path;C:\Ruby27", "User")
  • Locate and run the terminal program. Type 'irb' and hit enter.
